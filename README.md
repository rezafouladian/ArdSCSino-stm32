# FreeSCSI

This page is a work in progress.

ArdSCSino-stm32 とは たんぼ（TNB製作所）(https://twitter.com/h_koma2) さんが作成した ArdSCSino のSTM32版です<br>
ArdSCSino とは SCSIデバイス（ハードディスク）を arduino で再現するハードウエアです。<br>
許可を頂いて公開することになりました。<br>
